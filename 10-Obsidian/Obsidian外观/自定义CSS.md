# 自定义 CSS

如果你正在设计你个性化的 CSS，你可以通过使用开发者工具来获得更多元素信息，提高设计效率。使用开发者工具，1）在 Windows 或 Linux 下，使用快捷键 `Ctrl+Shift+I`；2）在 macOS 下，使用快捷键 `Cmd+Opt+I` 即可。

对于 CSS 的修改，Obsidian 是自动检测并会立即应用的。你不需要重启 Obsidian 来查看修改效果。
